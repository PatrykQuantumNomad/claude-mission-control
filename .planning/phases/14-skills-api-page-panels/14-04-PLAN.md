## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Backend response → React render | Decimal cost_usd as JSON string is never parsed as number for display (only for chart axis where precision loss is acceptable). |
| Backend SSE payload → SkillTimeline | attrs_skill_name from upstream OTLP body. React's text-content escaping handles XSS. |
| User filter input → events.filter | Pure client-side string match; no SQL/eval. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-14-04-01 | Tampering | cost_usd Decimal-as-string display | mitigate | Per D-02 / Pitfall 5: panel never calls Number(cost_usd) for the displayed dollar figure. Chart axis Number coercion is OK (recharts approximates anyway). |
| T-14-04-02 | Information Disclosure | SkillTimeline shows raw session_id values | accept | Single-user tool. Existing OtelPanel already shows session_id; no new disclosure surface. |
| T-14-04-03 | DoS | useFirehose buffer overflow on a chatty stream | mitigate | useFirehose has bufferSize prop (existing); SkillTimeline accepts the default cap (matches OtelPanel). |
