Imports:

State:
- `grabbed: boolean` — keyboard grab mode latch
- `announce: string` — aria-live region content

- Optional `gripRef` — focus the grip on grab/drop transitions to keep keyboard flow consistent

Handlers (each as a useCallback keyed on its dependencies):
1. `handleDragStart(e)`: write `e.dataTransfer.setData('text/cmc-panel-id', panelId)` + `e.dataTransfer.setData('text/cmc-column-id', columnId)` + `e.dataTransfer.effectAllowed = 'move'`. Optionally add class `cmc-panel--dragging` to the wrapper via setState; remove in `handleDragEnd`.
2. `handleDragOver(e)`: if `e.dataTransfer.types.includes('text/cmc-column-id')` → `e.preventDefault()` and `e.dataTransfer.dropEffect = 'move'`. Otherwise leave alone — irrelevant payloads are not droppable.
3. `handleDrop(e)`: `e.preventDefault()`; read fromId + fromColumn from dataTransfer; if `fromColumn !== columnId` → no-op (cross-column rejected per LAYO-02 single-column-only). Otherwise call `onReorder(fromId, index)`. Reset announce + grabbed state.
4. `handleKeyDown(e)`: branching per behavior block above. Use `e.preventDefault()` on every handled key to suppress browser-default scroll (especially ArrowDown). On Escape: setGrabbed(false), setAnnounce('Reorder cancelled.').

Render shape:
```jsx
<div
  className={`cmc-draggable-wrap ${grabbed ? 'cmc-draggable-wrap--grabbed' : ''}`}
  data-panel-id={panelId}
  data-column-id={columnId}
  onDragOver={handleDragOver}
  onDrop={handleDrop}
>
  <button
    ref={gripRef}
    type="button"
    draggable
    aria-label={`Reorder ${label}`}
    aria-pressed={grabbed}
    data-testid={`panel-drag-grip-${panelId}`}
    className="cmc-panel-grip"
    onDragStart={handleDragStart}
    onDragEnd={handleDragEnd}
    onKeyDown={handleKeyDown}

    <GripVertical size={14} aria-hidden />
  </button>
  <div role="status" aria-live="polite" className="cmc-sr-only">{announce}</div>
  {children}
</div>
```

```tsx
const pathname = useRouterState({ select: (s) => s.location.pathname })

const { isHidden, orderedPanels, setOrder } = useLayoutState(route)

// Build a Map<panelId, JSX> of every panel that could render in 'main'

  'cost-forecast': <PanelCard panelId="cost-forecast" headerMenu={...} ...>...</PanelCard>,
  'cost-by-project': <PanelCard panelId="cost-by-project" headerMenu={...} ...>...</PanelCard>,
}

// Render in the URL-driven order, wrapped in DraggablePanelWrap, filtered by isHidden
const mainOrder = orderedPanels('main')
const visibleMainPanels = mainOrder.filter(id => !isHidden(id) && id in mainPanels)
const total = visibleMainPanels.length

return (
  <div data-column-id="main" data-testid="panel-grid-main" className="cmc-card-grid">
    {visibleMainPanels.map((id, idx) => (
      <DraggablePanelWrap
        key={id}
        panelId={id}
        columnId="main"
        label={getPanelLabel(route, id)}
        index={idx}
        total={total}
        onReorder={(fromId, toIndex) => {
          const next = [...visibleMainPanels]
          const from = next.indexOf(fromId)
          if (from === -1 || from === toIndex) return
          const [moved] = next.splice(from, 1)
          next.splice(toIndex, 0, moved)
          setOrder('main', next)
        }}
      >
        {mainPanels[id]}
      </DraggablePanelWrap>
    ))}
  </div>
)
```
