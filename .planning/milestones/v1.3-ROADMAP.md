## Overview

Rebuilt the dashboard's UX from the ground up — fixed the three named daily-use friction modes (panels exceeding viewport, Sheets/Popovers escaping parent bounds via Radix Portal containment, data overflowing card edges via global `min-width: 0` on `.cmc-card`), shifted the aesthetic from IDE-references (Linear/Raycast/Vercel) to dashboard-product references (Honeycomb/Datadog/PostHog/Grafana family) via a new sidebar shell + 3-tier density toggle + global time picker, and unlocked four targeted new capabilities (3-tier density, server-persisted saved views with edit-vs-fork semantics, lightweight customizable layouts via show/hide + 1D reorder + split-pane, time-anchored navigation across every route) — all without breaking a single existing URL, API contract, or test suite. APPEND-ONLY `validateSearch` was enforced via a `tests/test_url_contract.py` + `docs/url-contract.md` gate (Pitfall 13 lock); the `data-testid` registry + ESLint custom rule kept Playwright selectors stable across a 461-test-net-add vitest suite and a 13→320 Playwright count growth. Every phase closed behind a formal visual checkpoint matrix (138 PNGs total across 5 phases × 3 densities × 2 themes) + axe-core gate + Lighthouse 9/9 baseline + perf budget. Three v1.2 carried tech debts closed cleanly inside Phase 27 (`project_key` wire exposure, `KNOWN_METRICS` removal + runtime API contract test, NL composer 503 retry UX honoring the V11 collapsed-failure-mode lock). One Alembic migration (`0004_saved_views`); zero new Python dependencies; 6 new runtime frontend deps locked at exact pins (`@radix-ui/react-popover@1.1.15`, `@radix-ui/react-dropdown-menu@2.1.16`, `sonner@2.0.7`, `react-day-picker@10.0.0`, `date-fns@4.1.0`, `react-resizable-panels@4.11.0`); zero new top-level routes (all five Phase 27 tail routes already existed). All milestone-close gates green: backend pytest 690/0/0 (vs Phase 18 baseline 566; +124), frontend vitest 754/0/0 (vs 293; +461), Playwright 320 tests in 19 spec files (vs 7+1-skipped; +313), Lighthouse 9/9 PASS at median (LCP 559-586ms, CLS 0-0.003, performance 1.0), axe 0 phase-attributable violations across 13 close-gate scans via PHASE_24..28 NET_CLASS_MARKERS inversion filter.
