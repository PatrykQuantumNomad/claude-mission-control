**Resolution:** Defer to a dedicated test-stabilization plan; not a v1.1 deliverable.

### Phase 15 — REQUIREMENTS.md ALRT-01/02 Markers

Both tables (`alert_rules`, `alert_state`) live in the codebase since Phase 13 migration 0002 and are exercised by 77 passing tests.

**Resolution:** Cosmetic doc cleanup — flip `[ ]` to `[x]` in a 2-line edit if desired.

### Phase 16 — `cmc/cost/engine.py` Reference Drift

**Resolution:** Update REQUIREMENTS.md citation if a future cleanup pass touches the file.

**Resolution:** Update test selector to `page.getByRole('form').getByLabel('Name')` in a future polish pass; not a v1.1 deliverable.
